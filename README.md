# AlexSolutions - Control de Inventario y Ventas 🚀

Sistema profesional desarrollado en Python para la gestión de salida de mercancía, liquidación de rutas y seguimiento de ventas mensuales.

## 📋 Requisitos del Sistema
Este programa requiere **Python 3.12** o superior instalado en el equipo.

## 🛠️ Instalación en un nuevo PC
Para ejecutar este programa en otro computador, sigue estos pasos:

1. **Clonar o descargar** el repositorio en una carpeta local.
2. **Instalar las librerías necesarias** ejecutando el siguiente comando en la terminal:
   ```bash
   pip install customtkinter pandas openpyxl fpdf

   Método 2: Clonar con Git (Recomendado para Alex)
Este método vincula tu PC con GitHub, permitiéndote bajar actualizaciones fácilmente después.

Instala Git: Descarga e instala Git para Windows en el nuevo PC.

Copia el enlace: En el botón "<> Code" de tu repositorio, copia la URL que aparece (ej. https://github.com/Alexander71133/AlexSolutions_Control.git).

Abre la Terminal: Ve a la carpeta donde quieras guardar el proyecto, haz clic derecho y selecciona "Open Git Bash here" o abre el CMD.
